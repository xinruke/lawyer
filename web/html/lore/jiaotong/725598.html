交通事故逃逸保险公司会依法赔偿吗 作者：未知     时间：2019-02-22    来源：互联网

　　在交通事故逃逸局面中，是否满足协议规定的赔偿认定条件呢？据小编了解，保险公司一般都会答应赔偿的。

　　交强险

　　1. 只要是被保险机动车遭遇了道路交通事故，导致本车人员、被保险人之外的受害人人身伤亡、财产损失的，由保险公司在法定的机动车交通事故责任强制保险责任之限额范围内，依法进行赔偿。

　　2. 保险公司在机动车交通事故责任强制保险责任限额范围内，对抢救费用有义务垫付，当然有权向致害人追偿。但是，若是以下这三种情况：

　　A  驾驶人未曾取得驾驶资格或醉酒；

　　B  被保险机动车于被盗抢期间发生交通肇事；

　　C  被保险人故意制造道路交通事故。

　　属于前款所列情形之一，发生道路交通事故，造成受害人财产损失的，保险公司不承担赔偿责任。

　　3. 交强险的免责范围

　　（1）道路交通事故损失由受害人故意造成，保险公司概不赔偿；

　　（2）下面四种损失和费用，交强险同样概不赔偿和垫付：

　　A  被保险人所有的财产及被保险机动车上的财产遭受的损失；

　　B  被保险机动车发生交通事故，致使受害人停业、停驶、停电、停水、停气、停产、通讯或者网络中断、数据丢失、电压变化等造成的损失以及受害人财产因市场价格变动造成的贬值、修理后因价值降低造成的损失等其他各种间接损失；

　　C  因交通事故产生的仲裁或者诉讼费用以及其他相关费用。

　　很明显，交通肇事逃逸不在交强险的免责范围。因此，交通肇事逃逸后，保险公司是要在交强险的范围内理赔的。

　　第三者商业责任险

　　第三者商业责任险属于商业险，是交强险之补充。交通安全法规定，交通事故损害纠纷发生后，应依照保险合同约定，确定保险公司承担的赔偿责任。

　　根据机动车商业保险行业基本条款，事故发生后，被保险人或其允许的驾驶人在未依法采取措施的情况下驾驶被保险机动车或遗弃被保险机动车逃离事故现场，或故意破坏、伪造现场、毁灭证据，保险人不负责赔偿。

　　其实就是说，交通肇事逃逸是汽车商业险的免责事由，保险公司可以在第三者商业责任险的范围内免责。

　　说了这么多，就是一点而已：交通肇事逃逸保险赔偿是有戏的。
